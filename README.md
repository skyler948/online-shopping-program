# Online Shopping Program
An online shopping program made for our CSCI 185 Computer Programming II Final
## Project Goal
Provide an easy-to-use interface for buying & shipping products online, intended for anyone who needs an online shopping experience.
## Key Concepts
- Inheritance
- Polymorphism
- Encapsulation
- Abstraction
- File Handling
- GUI using Swing
## Compiling Instructions
If you are using IntelliJ IDEA, you can easily import it by using the "Clone Repository" button in IntelliJ's project selection menu. You can then run the program from there.

If you are not using IntelliJ IDEA, make sure the `res` folder is present and in Java's build path.

This project requires no outside dependencies.
